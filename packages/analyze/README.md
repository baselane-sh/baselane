# @baselane/analyze

Static repo analysis (languages, frameworks, build/test commands, conventions, design
tokens) that powers the [`baselane`](https://www.npmjs.com/package/baselane) CLI's audit
and pack rendering.

## Install

```sh
npm install @baselane/analyze
```

Used as a library dependency of `baselane` — most users won't install it directly.

Docs: https://baselane.sh
