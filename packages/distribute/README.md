# @baselane/distribute

GitHub REST PR distribution and git-ref pack resolution (`github:owner/repo@ref`) for
[`baselane`](https://www.npmjs.com/package/baselane) workflow packs.

## Install

```sh
npm install @baselane/distribute
```

Used as a library dependency of `baselane` — most users won't install it directly.

Docs: https://baselane.sh
